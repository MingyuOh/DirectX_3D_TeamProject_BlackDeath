## 📝 개요
<img src="https://user-images.githubusercontent.com/16281218/219325279-37a2f698-fb7e-4784-8586-9faa801c27b6.png" width="800" height="600"/>    

> - 프로젝트: DirectX 11 기반 3D FPS Game BlackDeath     
> - 기획 및 제작: 오민규외 4명(클라이언트 4명, 기획 1명)   
> - 분류: 팀 프로젝트    
> - 담당 업무: 이펙트, 충돌 처리, 스킬 구현, 최적화 및 게임 시스템 개발    
> - 제작 기간: 60일    
> - 게임 소개:   
> &emsp;황폐해진 도시에서 바닥에 존재하는 아이템을 습득한 후 좀비들을 사살하며 생존하는 게임이다.    
> &emsp;총 네 개의 스테이지가 존재하며 제한 시간 내에 모든 좀비를 사살하지 못한다면 미션을 실패하게 되어 게임이 종료된다.    
> &emsp;한 개의 스테이지를 클리어하였을 때, 다음 스테이지 영역으로 이동하지 않는다면    
> &emsp;자기장에 의해 플레이어의 HP가 감소되므로 반드시 다음 스테이지 영역으로 이동해야 한다.   

## 🏆 목표      
- 4명의 프로그래머와 1명의 기획자가 팀을 이루어 DirectX를 활용하여 3D 게임을 제작한다.
- 각 클라이언트 개발자들은 자신이 맡은 파트(이펙트, 맵, 캐릭터, UI)의 툴을 이용하여 기획자가 요구한 사양(리소스 요청, 개발 기한 등)을 구현하여 게임 프로젝트에 적용한다.
- 팀 프로젝트를 진행함으로써 다른 작업자들과의 협업을 통해 개발하는 방법을 배우고 의사소통 능력을 향상시킨다.
